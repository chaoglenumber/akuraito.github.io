Hello World

发表于 2018-11-11

前言

　　经历了jimdo，手打GitHub和WordPress之后，终于还是换上了hexo，可以看出来博客有着正在向着轻量化发展的趋势，不过随之而来的，是搭建博客所需的基础更高了，虽然各家（指 WordPress 和 hexo）都有自己的开发文档可以帮助你在什么都不懂的情况下快速入门，但不可否认的是，门槛确实在不停地提高。

　　当然也有Bitcorn这样只用设置一次就 OK 的替代品，不过人家在内测，而且我也没有电脑，遂放弃，继续我的 hexo 之旅。

　　随着手机性能的不断提升，手机编程成为了可能，在此之前，我的 jimdo，GitHub 和 WordPress 都是在手机上完成的，虽然这并不算编程，不过我一直都在致力于提升 姿势 知识水平，希望能开发出新的技能，在一番误打误撞之后，发现了Termux。

Termux

　　目前对我来说只有一个作用：hexo，虽说应该算做是大材小用了，不过对于一个起步非常晚的高中生来说，还是写好自己的 hexo 就 ok。

　　Termux 对我来说就是一个 Linux 模拟器，当然也有其他的终端可供选择，不过由于 Termux 入门，所以对 Termux 有着天生的好感。

简记我的网站搭建历程

发表于 2018-10-28 | 更新于 2018-11-11

人生的第一个网站：cjlh.jimdo.com（长时间未使用已被回收）

　　应该是初中毕业之后逛 b 站的兴趣圈发现了 jimdo 这个东西，据说之前和淘宝有过合作，不过怎么看起来都不像的样子，毕竟连中文的操作页面都没有。🤣，现在不止是网站没有了，连 b 站的兴趣圈都没有了。

　　刚开始搭网站的时候完全不知道该弄些啥，放了几张图片（班主任的盛世美颜）之后就没怎么管它了，方便骗同学的点击。当时还把网址给了一个实习老师看 2333，我还记得那个老师还欠我一张明信片🤔。

第二个网站：nexguard.jimdo.com

　　从高二某个半期考试之后开始在上面写周记，应该只写了 37 篇，当然还有一篇 to buy list，两篇软件分享，还有一堆被删掉了的 blog。

　　之后又在酷安认识了 fa，就开始对 jimdo 造作，删 class 不亦乐乎，但不知道是因为 8.1 的锅还是 fa 的锅，在我的手机上经常闪退，目前看来是 jimdo 的锅，准备换一个简单点的网站，于是就有了后来的 GitHub。

第三个网站：nexguard.github.io

　　其实很早之前就注册过 GitHub，当时拿着 Nexguard 这个名字到处注册/改名字（酷安，网易云邮箱，网易云音乐，微博，bilibili，手机乐园，知乎，GitHub 成就已达成），但有一个很重要的问题，我不知道要怎么写网页啊，遂进 data/data 找到了 via 浏览器的 about.html，通过对比在浏览器上显示的网页和文件源码搞清楚了

的作用，真是一个励志的故事.jpg，然后通过 fa 来代替导航栏，以挽救我那差的不行的 html 水平。

　　也去找过一堆主题，不过没有电脑，直接就费掉了 orz。

然后就是现在的这个 WordPress 啦（并不）

　　现在用的是乐云的白嫖服务器，时不时会断一下，一般从周六下午开始，有钱真好.jpg。

现在在玩 hexo，所以把这篇文搬过来。
